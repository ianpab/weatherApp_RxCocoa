# Weather App

# Sobre

Projeto desenvolvido pelo site <a href="https://www.raywenderlich.com" target="_blank">
raywenderlich.com</a> sobre como utilizar RxCocoa e suas diversas funcionalidades.<br>
No aplicativo é possivel pesquisar o tempo através de uma cidade no campo de texto, através de sua localização<br>
ou navegando pelo mapa, onde é exibido automaticamente o tempo da região na qual está apontando.
